# 🚀 START HERE

## Quick Start (3 Steps)

### 1. Install Dependencies (if not done)
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open Browser
- **Main App**: http://localhost:3000
- **Test Page**: http://localhost:3000/test
